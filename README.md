# Full-Stack Blackjack Game

A secure, full-stack Blackjack game built with React, Node.js, and SQL. This project is an updated version of my original JavaScript/HTML Blackjack game, featuring encrypted login, persistent chip tracking, and an improved UI.

---

## Description

This full-stack Blackjack app allows users to register, log in securely, and play Blackjack against a dealer. The app features:

- Secure user registration and login with encrypted passwords 
- Real-time game updates with React frontend  
- Persistent chip tracking using SQL  
- Updated UI and gameplay compared to the original JavaScript/HTML version  

---

## Features

- User authentication with bcrypt password hashing  
- Single-player gameplay against dealer  
- Persistent chip tracking in SQL database  
- Responsive UI built with React  
- Modern game logic with React state management  

---

## Tech Stack

- **Frontend:** React, HTML, CSS, JavaScript  
- **Backend:** Node.js
- **Database:** SQL 
- **Authentication:** bcrypt for password hashing
